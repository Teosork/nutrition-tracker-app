# Nutrition Tracker App

A web application that helps users track their daily intake by using product barcodes and nutrition data from Open Food Facts.

## Project Goal

The goal of this project is to build an app that fetches nutrition data for the provided product barcodes and displays nutrition values per product, total meal values, and how the meal fits into daily nutrition targets.

## MVP

- Display the provided products using their barcodes
- Show Kcal, Fat, Carbs, and Protein for each product
- Calculate total nutrition values for the full meal
- Compare meal totals against daily nutrition targets
